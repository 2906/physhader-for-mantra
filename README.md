# PhyShader

Physical plausible, easy to use, compact surface shader for Mantra renderer.

![PhySurface Materials](img/materials.jpg "Materials preview")

## Features
* VOP type operator
* Energy conserving
* Microfacet-based BSDF's
* Anisotropy
* Conductor Fresnel
* Volume absorption
* Subsurface scattering
* Translucency
* Dispersion
* Thin plate shading
* Transparent shadows
* Shader nesting
* Extra image planes support
* and more...

##Installation

### On Linux
```
make
make install
```

### On Windows
Run `install.bat`
